# Learning JavaScript 🚀

A structured collection of JavaScript exercises and concepts — from basics to loops. Each chapter builds on the previous, covering core JS topics through hands-on coding examples.

## 📚 Chapters

| Chapter | Topic | Files |
|---------|-------|-------|
| **Chapter 1** | Basics | Variables, console, setup |
| **Chapter 2** | JavaScript Intro | JS basics |
| **Chapter 3** | Identifiers & Literals | Identifier rules, naming |
| **Chapter 4** | JS Concepts | `var`/`let`/`const`, hoisting, scope, functions |
| **Chapter 5** | Literals | String, number, template literals, `null`/`undefined` |
| **Chapter 6** | Operators | Arithmetic, comparison, logical, ternary, increment, nullish |
| **Chapter 7** | If/Else | Conditionals, grade calculator, leap year, even/odd |
| **Chapter 8** | Switch | Switch statements, grouping, real-world examples |
| **Chapter 9** | User Input | `prompt-sync`, Node `readline` |
| **Chapter 10** | Loops | `for`, `while`, `do-while`, `for...of`, `for...in` |
| **Chapter 11** | Arrays | Array methods, iteration, manipulation |
| **Chapter 12** | Functions | Function declarations, expressions, arrow functions |
| **Chapter 13** | Strings | String methods, template literals |
| **Chapter 14** | Objects | Object literals, properties, methods |
| **Chapter 15** | 2D Arrays | Nested arrays, matrix operations |
| **Chapter 16** | Callbacks | Callback functions, sync/async callbacks, callback hell |
| **Chapter 17** | Promises | Promise states, chaining, error handling, Promise.all |
| **Chapter 18** | Async/Await | async/await syntax, error handling, parallel execution |
| **Chapter 19** | Playwright Basics | Browser automation setup, basic navigation |

## 🛠️ Languages

- JavaScript (Node.js)

## 🚦 Getting Started

```bash
# Clone the repo
git clone https://github.com/evelynsini/Learning.git

# Navigate into a chapter
cd Learning/Chapter10_Loops

# Run a file with Node
node 71.ForLoop1.js
```
